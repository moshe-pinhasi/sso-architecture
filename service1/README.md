## Docker command
#### Build

```
$ docker build .
```

### Build with tag
```
$ docker build -t sso-example/service1 .
```

### Run using id
```
$ docker run :id
```

### Run using tag
```
$ docker run sso-example/service1
```

### List all running dockers
```
$ docker ps
```

### Stop runing docker
```
$ docker stop :container_id
```

### Docker log
```
$ docker logs :container_id
```
